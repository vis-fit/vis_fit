- Iniciamos a construção de um site.
- O nome do site é "VIS" e para referencia o nome do projeto é "vis_fit"
- Estamos usando o VSCode
- O backend é em node.js e está hospedado no Render conectado ao front que está no GitHub
- O banco de dados está no Neon.tech
- Já conectamos o banco ao site
- Já criamos a tela principal e as páginas dos módulos
- As páginas dos módulos contam apenas com o header e o footer
- Tudo que vamos desenvolver nas páginas será no corpo
- Vamos iniciar o desenvolvimento do modulo de alimentos
- O modulo de alimentos será na página mod_food.html
- Para essa página, quero um arquivo CSS separado
- Para essa página, quero um arquivo js separado (as partes que forem exclusivas)
